# 工业产品国内发布：无备案宣传与演示环境构建方案

> 更新日期：2025-07-12
> 适用场景：工业产品（SaaS/软硬件）尚未完成 ICP 备案，需要在国内快速开展宣传、构建演示环境和官网

---

## 重要前提：国内可访问的平台现状（2025年）

| 平台 | 国内访问 | 是否需要备案 | 适合用途 |
|------|---------|-------------|---------|
| **Vercel** (`*.vercel.app`) | ❌ DNS被污染，极不稳定 | 不需要 | ~~不推荐~~ |
| **Cloudflare Pages** | ❌ 基本无法访问 | 不需要 | ~~不推荐~~ |
| **GitHub Pages** | ⚠️ 时好时坏，经常超时 | 不需要 | ~~不推荐用于正式产品~~ |
| **腾讯云 EdgeOne Pages（国际版）** | ✅ 海外CDN节点，国内可流畅访问 | 不需要 | ✅ 静态官网 |
| **帽子云 (maoziyun.com)** | ✅ 国内CDN节点，速度快 | 不需要 | ✅ 静态官网 |
| **香港云服务器** | ✅ 延迟30-80ms，非常稳定 | 不需要 | ✅ 官网+演示后端 |
| **微信生态（公众号/小程序）** | ✅ 微信内完美访问 | 不需要ICP（需微信认证） | ✅ 获客主阵地 |

> **结论：Vercel / Cloudflare / GitHub Pages 在国内都不可靠，不能作为工业产品的正式部署方案。**

## 二、方案详解

### 方案A：香港云服务器全栈部署（强烈推荐）

**这是最靠谱的方案：无需备案，国内访问快，可部署前后端完整服务。**

#### 部署架构

```
客户浏览器/微信
       ↓
  自有域名 (如 product.com)
       ↓
  DNS 解析 → 香港服务器 IP
       ↓
  ┌──────────────────────────┐
  │     香港轻量云服务器       │
  │  (阿里云/腾讯云 香港节点)  │
  ├──────────────────────────┤
  │  Nginx (反向代理+SSL)     │
  │  ├─ 前端静态站 (官网)      │
  │  ├─ 后端 API 服务          │
  │  └─ Docker 容器化部署      │
  └──────────┬───────────────┘
             ↓
  ┌──────────────────────────┐
  │     数据库                │
  │  (同服务器 Docker 或       │
  │   阿里云 RDS 香港版)       │
  └──────────────────────────┘
```

#### 具体步骤

```bash
# 1. 购买香港服务器
# 推荐：阿里云香港 轻量应用服务器 2核4G 60GB SSD
# 价格：约 ¥70-100/月

# 2. 一键部署环境
curl -fsSL https://get.docker.com | bash
docker-compose up -d

# 3. 配置 Nginx 反向代理 + Let's Encrypt SSL
# 4. 域名解析到香港服务器IP
# 5. 完成！国内访问延迟 30-80ms
```

#### docker-compose.yml 示例

```yaml
version: '3.8'
services:
  # 前端官网
  web:
    build: ./frontend
    ports:
      - "80:80"
    depends_on:
      - api
    restart: always

  # 后端 API（演示环境）
  api:
    build: ./backend
    ports:
      - "3000:3000"
    environment:
      - DATABASE_URL=postgresql://user:pass@db:5432/demo
    depends_on:
      - db
    restart: always

  # 数据库
  db:
    image: postgres:16-alpine
    environment:
      POSTGRES_DB: demo
      POSTGRES_USER: user
      POSTGRES_PASSWORD: pass
    volumes:
      - pgdata:/var/lib/postgresql/data
    restart: always

volumes:
  pgdata:
```

#### 推荐配置与成本

| 组件 | 推荐 | 月成本 |
|------|------|--------|
| 服务器 | 阿里云香港 轻量 2核4G | ~¥70-100 |
| 域名 | .com 域名 | ~¥5/月（60元/年） |
| SSL 证书 | Let's Encrypt | 免费 |
| 数据库 | Docker 自建 PostgreSQL | 含在服务器内 |
| **合计** | | **~¥75-105/月** |

#### 优点
- **国内访问极快**：香港到大陆延迟仅 30-80ms
- **完全无需 ICP 备案**
- **可部署完整前后端**：演示环境需要数据库、API 都没问题
- **技术栈无限制**：任何语言、任何框架都能跑
- **稳定可靠**：阿里云/腾讯云香港机房非常稳定

#### 缺点
- 需要基础 Linux/Docker 运维能力
- 极端情况下可能被墙（概率极低）

### 方案C：EdgeOne Pages / 帽子云 部署静态官网

**适合纯展示型官网（无后端需求），零运维，5分钟上线。**

#### 选项1：腾讯云 EdgeOne Pages（国际版）

```
1. 注册腾讯云国际站账号（仅需邮箱，无需银行卡）
2. 绑定 GitHub 仓库
3. 选择项目 → 自动构建部署
4. 绑定自有域名
5. 选择"全球可用区（不含中国大陆）"加速区域

**关键限制：** 未备案域名只能使用海外CDN节点，但国内仍可流畅访问。

| 特点 | 说明 |
|------|------|
| 支持框架 | React、Vue、Next.js、Hexo 等 |
| 部署方式 | GitHub 联动，自动构建 |

| 国内访问 | 海外CDN节点，流畅 |
| 费用 | 免费套餐可用 |

```
1. 注册帽子云账号
2. 绑定 GitHub/Gitee 仓库

| 特点 | 说明 |

### 方案D：同步启动 ICP 备案（长期必做）

**无论选择上述哪个方案，都建议同步启动备案。**

```
企业 ICP 备案流程：
1. 购买境内服务器（阿里云/腾讯云）
2. 通过服务商提交备案
3. 审核：1-20个工作日（各省份不同）
4. 通过后 → 境内CDN → 最佳体验
```

**2025年备案要点：**
- 企业主体：营业执照 + 法人身份证 + 负责人身份证
- 个人主体：**不可用于商业产品宣传**
- 备案期间：境外服务正常运营，不受影响
- 经营性服务（在线收费）：需额外申请 ICP 许可证（EDI）
